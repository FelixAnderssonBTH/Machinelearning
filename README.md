# Machine Learning

This repository contains three assignments focused on the application, evaluation, and implementation of traditional machine learning algorithms for classification, comparative analysis, and personalized recommendations.

---

## Assignment 1: Wine Quality Classification
The objective of this assignment was to train and evaluate two distinct classifiers to identify wine quality and determine the superior model.
* **Models Used:** Linear Discriminant Analysis and Random Forest.



---

## Assignment 2: Spam Detection Performance Analysis
A rigorous experimental comparison of three learning algorithms applied to a spam detection task.
* **Methodology:**
    * **Stratified Ten-Fold Cross-Validation** to ensure stable performance metrics.
    * **Statistical Significance:** Conducted the **Friedman test** to determine if the performance differences between the algorithms were statistically significant.

---

## Assignment 3: Personalized Fitness Training Schemes
The final project explores the application of traditional machine learning (excluding Deep Learning) to solve the real-world challenge of automated fitness programming.
* **The Goal:** To create an algorithm capable of generating personalized training schemes based on specific user goals, preferences, and experience levels.
* **Scope:** Designed to be accessible for everyone, ranging from complete beginners to experienced athletes.
* **Relevance:** Addressing the need for scalable, data-driven health and industry solutions.
